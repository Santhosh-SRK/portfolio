# Santhosh Portfolio

Personal frontend portfolio built with React, TypeScript, MUI, and Framer Motion.

## Live Site

GitHub Pages:
`https://santhosh-srk.github.io/portfolio/`

## Stack

- React
- TypeScript
- MUI
- Framer Motion
- Vite
- Storybook

## Scripts

```bash
npm install
npm run dev
npm run build
npm run storybook
```

## Deployment

This repo is configured for GitHub Pages using GitHub Actions.
Every push to `main` triggers a production build and Pages deployment.
